# Vishay Intertechnology

Vishay Intertechnology is one of the world's largest manufacturers of discrete semiconductors (diodes, rectifiers, MOSFETs, optoelectronics, selected ICs) and passive electronic components (resistors, inductors, capacitors). Vishay components are used in automotive, industrial, computing, consumer, telecommunications, military, aerospace, and medical applications.

**Website:** https://www.vishay.com/  
**Product Selector:** https://www.vishay.com/en/product-selector/  
**Document Library:** https://www.vishay.com/en/how/onlineliterature/online-libraries/

## Data Access

Vishay does not provide a public REST API directly. Component data is accessible via:

- **Octopart/Nexar API:** https://nexar.com/api — Search Vishay parts, compare pricing, and retrieve datasheets
- **DigiKey API:** https://developer.digikey.com/ — Vishay inventory, pricing, and specifications
- **Vishay Parts Library (VPL):** Cross-reference tool for finding Vishay alternatives
